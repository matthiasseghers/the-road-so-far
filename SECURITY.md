# Security Policy

## Scope

The Road So Far is designed for **local, single-user use**. The Express API
has no authentication and should never be exposed to the public internet.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

1. **Do not** open a public issue.
2. Use [GitHub's private vulnerability reporting](https://github.com/matthiasseghers/the-road-so-far/security/advisories/new).

You should receive an initial response within 72 hours. Once confirmed, a fix
will be released as soon as possible and you will be credited in the release
notes (unless you prefer to stay anonymous).

## Supported Versions

Only the latest version on `main` is supported. There are no backport branches.
